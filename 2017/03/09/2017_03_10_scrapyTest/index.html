1.1. 环境信息

用 windows 系统搭建scrapy环境跟麻烦，由于scrapy需要依赖twisted，而twisted只有源码编译，如果系统安装时没有合适的编译环境，则不能正常安装。
　 　于是选择linux作为系统环境，比较方便搭建。环境详情如下：
