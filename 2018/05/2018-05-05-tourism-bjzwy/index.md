# 18年5月5日游北京植物园


　　今天水一次，去了趟北京植物园，天气不是特别好，重度污染，下点小雨，不过人相当多！！！
　　*第一次上传照片太大了，平均3M往上，打开网页都费劲，后来把照片压缩了一下，才好些*

　　园子里很多花都开了，只顾了花拍照了，忘了记下名字，以下花的名称都是识别软件识别的，不保证准确…………
　　从南门进去的，今年没买年票，得临时掏腰包了，还好支持交通卡入园，不用去窗口排队。最重要的是去公园买一张票总是收到售票员奇怪的目光。
